Для некоммерческого использования


## Run Locally

Clone the project

```bash
  git clone https://github.com/dustin1192/rpw-phpbb.git
```

Go to the project directory

```bash
  cd bot
```

Install dependencies

```bash
  npm i
```


Create file .env

```DISCORD_TOKEN=
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
TABLE_PREFIX=
```

Start the server

```bash
  npm run start
```
